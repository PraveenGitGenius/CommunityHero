# Community Hero

An AI-powered civic issue reporting and monitoring platform that enables citizens to report community problems such as potholes, garbage, water leakage, streetlight failures, flooding, and more using image-based analysis powered by Google Gemini AI.

---

## Features

- AI-powered issue detection using Google Gemini
- Image upload and analysis
- Geolocation support
- Interactive Map Visualization
- Real-time Dashboard
- Analytics and Charts
- Leaderboard & Gamification
- Community Verification
- Automatic Complaint Generation
- Search & Filter Reports
- Live Issue Feed

---
